For Windows Users:

  1. Open Task Manager (Ctrl + Shift + Esc)

  4. Close all browser tabs except the interview page
  5. Refresh the interview page (F5)

  3. Ensure it's set to "Allow"
  4. Refresh the page after changing permissions
