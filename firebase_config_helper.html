Step 1: Get Your Firebase Configuration

Click the button below to open your Firebase project settings:

Open Firebase Settings

Scroll down to "Your apps" section and look for your web app configuration.

Step 2: Copy Your Configuration

You should see a code snippet like this:

const firebaseConfig = { apiKey: "AIzaSy...", authDomain: "mokmat-c3474.firebaseapp.com", projectId: "mokmat-c3474", storageBucket: "mokmat-c3474.appspot.com", messagingSenderId: "123456789", appId: "1:123456789:web:abc123", measurementId: "G-ABC123" };

Copy all the values from your Firebase console.

Step 3: Update Your Configuration File

Open the file: static/js/firebase_auth.js

Replace lines 7-14 with your actual values:

const firebaseConfig = { apiKey: "YOUR_ACTUAL_API_KEY_HERE", authDomain: "mokmat-c3474.firebaseapp.com", projectId: "mokmat-c3474", storageBucket: "mokmat-c3474.appspot.com", messagingSenderId: "YOUR_ACTUAL_SENDER_ID", appId: "YOUR_ACTUAL_APP_ID", measurementId: "YOUR_ACTUAL_MEASUREMENT_ID" };

Step 4: Enable Google Authentication

Click the button below to enable Google Sign-In:

Enable Google Sign-In

1. Click on "Google" provider
2. Toggle "Enable"
3. Select your support email
4. Click "Save"

Step 5: Create Firestore Database

Click the button below to create your database:

Create Firestore Database

1. Click "Create database"
2. Select "Start in test mode"
3. Choose location (e.g., asia-south1 for India)
4. Click "Enable"

Step 6: Add Authorized Domain

Add localhost to authorized domains:

Add Authorized Domains

1. Go to "Authorized domains" tab
2. Add: localhost
3. Click "Add domain"
